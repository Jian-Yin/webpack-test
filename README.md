# webpack-test

一个基于 webpack + gulp + handlebars 的项目结构构建；

gulp 实现自动化；
webpack 打包工具实现按需加载；
handlebars 模板实现 html 复用以及数据嵌入，能实现组件化

初始化项目：npm install
打包项目: npm run build
运行项目: npm run start
